# The Mulligan Report — Drudge Edition

A Drudge Report-style golf news aggregator with Augusta green + gold palette.

## Files

- `index.html` — Router. Detects mobile vs desktop and redirects automatically.
- `desktop.html` — 3-column Drudge layout for desktop browsers.
- `mobile.html` — Single-column feed optimized for phones.

## Deploy

Upload all files to the same directory on Vercel, GitHub Pages, or any static host.

## Features

- 3-column Drudge Report layout (desktop)
- Single-column smart feed (mobile)
- 27 clickable articles with full content overlay
- Article pages with subscribe CTA + Read Original links
- Scrolling leaderboard crawl ticker
- Sticky utility bar (Leaderboard, Tee Times, Odds, WITB, TV)
- Search box with live story filtering
- Tips/comment submission box
- 3 email opt-in placements per page
- Pro Pick sponsored review sections
- Hot takes with editorial voice
- Daily cartoon (The Rough Sketch)
- Golf history archive
- Podcast + source directories
- Visit counter
- App promo block

## Stack

Pure HTML/CSS/JS. No frameworks. No build tools. No dependencies.
